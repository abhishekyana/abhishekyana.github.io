My Projects

  1. Biometric Authentication System for Vehicles: Post Link

    1. Inspiration for this Project:

      2. One day, I misplaced my bike keys and had to wait for 1hr to get a duplicate key to again get back home, then I thought of a solution that could enable us to have a key that we'll never misplace, then after getting a bit of inspiration from Apple's TouchID, I decided upon using my fingerprint as the key to unlock the bike.

    2. Working:

      1. Fingerprint scanner that reads the user's fingerprint and unlocks the vehicle if the user is registered else, it'll capture the bitmap of the user if he/she tries to tamper the system, that'll help is getting the details of the person.
      2. A Biometric authentication system for the bike where a user can use his finger as a key to start the bike.
      3. When a registered user puts his finger, the fingerprint sensor verifies the saved IDs and the microcontroller unlocks the handle and gives ignition to the bike.
      4. This system also features an offline server where a user can lend the bike to an unregistered user using a password generated with the time seed.
      5. It can also monitor the bikes health and can warn the user if there is any malfunction in the bike.


  2. FaceID for MacBook Using HOGs and CNNs:Post Link

    1. Working:

      1. When any person tries to open my private projects, the python program will first capture the image from the FaceTime Camera.
      2. Applies HOGs (Histogram of Oriented Gradients) on the captured image to detect the bounding boxes of the face.
      3. The detected face is given to a CNN (Inspired by Google’s Facenet) where it outputs a 128 dimension feature vector of the face.
      4. The vector is then compared with the pre-stored face vectors using the cosine distance for similarity.
      5. If the image passes through the threshold (85%) then private files which are previously encrypted using Openssl aes-256 encryption are now decrypted.
      6. If the face in the image is not similar to pre-stored vectors then the program will send the image containing the face to my mail as an alert.


  3. Robotic Arm with Improved Dexterity Controlled with Speech Commands:Post Link

    1. Working:

      1. The python script converts the speech into text using cloud API.
      2. Then the text is analysed and converted into appropriate tasks for the hand to perform like open and close.
      3. Then the data is written onto the serial port of Arduino, then arduino will control the servos’ position depending upon the data received.
