  2. FaceID for MacBook Using HOGs and CNNs:Post Link

    1. Inspiration for this Project:

      2. Though I have projects that anybody can view, contribute and use. Some projects I have are really private for me. I have a friend who regularly challenges me by modifying some of my private projects. Though I’ve a password for my MacBook, Hiding that big thing and typing the password was cumbersome. Funnily enough, we were like GANs(kind of), I used to change the password regularly using different patterns and methods, He used to spy and open them. Then after thinking, I’ve implemented this FaceID Decrypter for certain specified files. This enabled me to protect my private projects, without having to fear about being teased. Because, my face is my password.

    2. Working:

      • When any person tries to open my private projects, the python program will first capture the image from the FaceTime Camera.
      • Applies HOGs (Histogram of Oriented Gradients) on the captured image to detect the bounding boxes of the face.
      • The detected face is given to a CNN (Inspired by Google’s Facenet) where it outputs a 128 dimension feature vector of the face.
      • The vector is then compared with the pre-stored face vectors using the cosine distance for similarity.
      • If the image passes through the threshold (85%) then private files which are previously encrypted using Openssl aes-256 encryption are now decrypted.
      • If the face in the image is not similar to pre-stored vectors then the program will send the image containing the face to my mail as an alert.

    2. Working:

      • The Speech is converted into text using CloudAPI. Internally it uses Connectionist Temporal Classification for transcribing the speech.
      • The intent and entity are detected using the Wit.ai’s NLP API.
      • The intent is recognized according to the given command, like greeting, recommending movies, OS control etc.
      • The response is generated from the pre-defined templates, the text is converted into speech using the native ‘say’ command in the MacOS.
      • Recommendation systems is built naively and bag-of-words model is used for Sentiment analysis to reduce the processing time.
